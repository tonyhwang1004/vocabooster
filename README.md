## 설치 방법 (GitHub Pages)

1. 이 저장소를 Fork 하거나 파일을 다운로드
2. GitHub에서 새 저장소 생성 (예: `vocabooster`)
3. 모든 파일 업로드
4. Settings → Pages → Source: `main` 브랜치 선택
5. 몇 분 후 `https://[username].github.io/vocabooster/` 에서 접속 가능

## 파일 구조

```
vocabooster/
├── index.html          # 메인 페이지
├── test.html           # 테스트 페이지
├── result.html         # 결과 페이지
├── ranking.html        # 시험 결과 목록
├── falling_bomb.html   # 단어 게임
├── student_analysis.html # 학습 분석
└── data/
    ├── books.json      # 교재 목록
    └── books/          # 교재별 단어 데이터
        ├── 능률기본/
        ├── 능률실력/
        └── ...
```
